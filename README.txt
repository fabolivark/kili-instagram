== Description ==

Generate instagram feed object. The plugin uses the instagram public api. Use the function get_instagram_feed() in ypur theme to get the object.

== Installation ==

This section describes how to install the plugin and get it working.

1. Download the latest release
2. Upload `kili-instagram.zip` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress

== Usage ==

1. Go to `Appearance -> Customize` or `Appearance -> Themes`, select your theme and click on `Customize`
2. Click on `Kili Instagram options -> Access token` and enter your token from instagram developer (https://www.instagram.com/developer/)
3. Use the function get_instagram_feed() in your theme to get the object :)
